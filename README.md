# Kinji Product Analytics Case Study

A single-page portfolio case study built for Product Analyst / Product Strategy applications.

## Files

- `index.html` — full case study website
- `vercel.json` — Vercel routing + security headers
- `README.md` — project overview

## Deploy to Vercel

1. Create a new GitHub repository.
2. Upload these files to the root of the repo.
3. In Vercel, choose **Add New Project**.
4. Import the GitHub repo.
5. Use the default settings.
6. Click **Deploy**.

## Positioning

This portfolio frames Kinji as a product analytics case study, not just an app idea. It highlights:

- KPI design
- Data instrumentation
- AI product logic
- User activation strategy
- Experimentation planning
- Monetization logic
